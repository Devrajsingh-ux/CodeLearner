# 🚀 Quick Start - Connect All Courses

## TL;DR

Get all 49 programming courses live on your website in 3 steps:

### Step 1: Start Dev Server
```bash
npm run dev
```

### Step 2: Login as Admin
Navigate to: `http://localhost:3000/admin/login`

### Step 3: Import Curricula
1. Go to: `http://localhost:3000/admin/courses`
2. Click: **"Import 49 Curricula"** (green button with sparkles ✨)
3. Confirm and wait (~60 seconds)
4. Done! ✅

## Troubleshooting

**Import button doesn't work?**
- Make sure you're logged in as admin
- Check browser console (F12) for errors

**No courses showing?**
- Refresh the page
- Check `/admin/courses` to see if import succeeded
- Verify Appwrite is configured

**Need help?**
- See `CONNECT_COURSES_GUIDE.md` for detailed guide
- See `CURRICULUM_INTEGRATION_SUMMARY.md` for technical details
